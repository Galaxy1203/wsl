# 量化投资工具 - 虚拟环境使用说明

## 环境状态
- ✅ 虚拟环境已创建: `quant_env`
- ✅ 依赖包已安装: numpy, pandas, tabulate
- ✅ 工具已测试可用

## 使用方法

### 1. 激活虚拟环境
```bash
cd /home/galaxy_61973/my_file
source quant_env/bin/activate
```

### 2. 运行量化工具
```bash
cd quant

# 交互模式
python3 quant_tool.py

# 测试单个策略
python3 quant_tool.py --strategy mean_reversion

# 自定义初始资金
python3 quant_tool.py --strategy sma_cross --initial-capital 200000

# 测试所有策略
python3 quant_tool.py --all

# 查看帮助
python3 quant_tool.py --help
```

### 3. 退出虚拟环境
```bash
deactivate
```

## 可用策略
- sma_cross      - 双均线交叉策略
- rsi            - RSI策略
- bollinger      - 布林带策略
- macd           - MACD策略
- mean_reversion - 均值回归策略
- momentum       - 动量策略
