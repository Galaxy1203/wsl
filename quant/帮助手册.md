## 工具简介

本量化投资工具是一个模拟的量化投资回测系统，包含以下功能：

- **数据生成**：生成模拟的股票价格数据
- **策略开发**：内置多种经典量化投资策略
- **回测引擎**：评估策略的历史表现
- **性能分析**：计算多种风险和收益指标

### 工具结构

```
quant/
├── data_processor.py    # 数据处理模块
├── strategies.py        # 策略模块
├── backtester.py        # 回测引擎
├── quant_tool.py        # 主程序
└── 帮助手册.md          # 本帮助文档
```

## 核心模块详解

### 1. 数据处理模块 (data_processor.py)

#### DataGenerator 类
用于生成模拟的市场数据。

**主要方法：**
- `generate_price_data()`: 生成单个资产的价格数据
- `generate_multiple_assets()`: 生成多个资产的价格数据

#### TechnicalIndicators 类
计算技术分析指标。

**支持的指标：**
- SMA（简单移动平均）
- EMA（指数移动平均）
- RSI（相对强弱指标）
- Bollinger Bands（布林带）
- MACD（异同移动平均线）

### 2. 策略模块 (strategies.py)

包含多种量化策略，所有策略都继承自基类 `Strategy`。

**核心方法：**
- `generate_signals(data)`: 生成交易信号
  - 返回值：1 = 买入，-1 = 卖出，0 = 持有

### 3. 回测引擎 (backtester.py)

#### Backtester 类
执行策略回测并计算性能指标。

**主要功能：**
- 模拟交易执行
- 计算组合价值变化
- 统计性能指标
- 记录交易明细

#### PortfolioOptimizer 类
投资组合优化（简化版）。

## 策略说明

### 1. 双均线交叉策略 (SMACrossStrategy)

**原理：**
当短期均线上穿长期均线时买入，下穿时卖出。

**参数：**
- `short_window`: 短期均线周期（默认：5）
- `long_window`: 长期均线周期（默认：20）

**适用场景：**
趋势市场效果较好，震荡市可能产生频繁交易。

---

### 2. RSI策略 (RSIStrategy)

**原理：**
当RSI低于超卖线时买入，高于超买线时卖出。

**参数：**
- `window`: RSI计算周期（默认：14）
- `oversold`: 超卖线（默认：30）
- `overbought`: 超买线（默认：70）

**适用场景：**
震荡市效果较好，单边趋势市可能过早离场。

---

### 3. 布林带策略 (BollingerBandsStrategy)

**原理：**
价格跌破下轨时买入，突破上轨时卖出。

**参数：**
- `window`: 均线周期（默认：20）
- `std_dev`: 标准差倍数（默认：2）

**适用场景：**
均值回归类策略，适合震荡市场。

---

### 4. MACD策略 (MACDStrategy)

**原理：**
MACD线上穿信号线时买入，下穿时卖出。

**参数：**
- `fast`: 快线周期（默认：12）
- `slow`: 慢线周期（默认：26）
- `signal`: 信号线周期（默认：9）

**适用场景：**
趋势跟踪策略，捕捉中期趋势。

---

### 5. 均值回归策略 (MeanReversionStrategy)

**原理：**
价格偏离均值一定幅度时买入，回归均值时卖出。

**参数：**
- `window`: 均值窗口（默认：20）
- `entry_threshold`: 入场阈值（默认：1.0标准差）
- `exit_threshold`: 出场阈值（默认：0.0标准差）

**适用场景：**
假设价格会回归均值，适合波动较大的品种。

---

### 6. 动量策略 (MomentumStrategy)

**原理：**
过去N天涨幅为正时买入，持有M天后卖出。

**参数：**
- `lookback`: 回看周期（默认：20）
- `hold_period`: 持有周期（默认：10）

**适用场景：**
追涨杀跌，适合趋势明确的市场。

## 性能指标解读

### 1. 收益率指标

- **总收益率**: 整个回测期内的累计收益
  - 计算公式：(最终价值 - 初始资金) / 初始资金

- **年化收益率**: 将总收益率转换为年度化收益
  - 考虑了复利效应，便于不同周期策略比较

### 2. 风险指标

- **波动率**: 衡量收益率的波动程度
  - 数值越大，风险越高
  - 计算为日收益率标准差的年化值

- **最大回撤**: 从最高点到最低点的最大跌幅
  - 衡量最坏情况下的损失
  - 计算公式：(组合价值 - 历史最高) / 历史最高

### 3. 风险调整收益指标

- **夏普比率**: 每承担一单位风险获得的超额收益
  - 计算公式：(年化收益率 - 无风险利率) / 波动率
  - 本工具假设无风险利率为0
  - 数值越大越好，一般 >1.0 被认为是良好

### 4. 对比指标

- **基准收益**: 买入并持有的收益率
- **超额收益**: 策略收益减去基准收益

### 示例3：在代码中自定义使用

创建一个新文件 `custom_example.py`：

```python
from datetime import datetime, timedelta
from data_processor import DataGenerator
from strategies import SMACrossStrategy
from backtester import Backtester

# 1. 生成数据
generator = DataGenerator()
end_date = datetime.now()
start_date = end_date - timedelta(days=365)
data = generator.generate_price_data(start_date, end_date)

# 2. 创建策略
strategy = SMACrossStrategy(short_window=10, long_window=30)
signals = strategy.generate_signals(data)

# 3. 运行回测
backtester = Backtester(initial_capital=200000, commission=0.0005)
results = backtester.run_backtest(data, signals)

# 4. 打印结果
backtester.print_performance_summary(results['performance'])
```

        return signals

## 注意事项

1. **模拟数据**: 本工具使用随机生成的模拟数据，不代表真实市场表现
2. **过拟合风险**: 历史表现不代表未来收益，避免过度优化历史数据
3. **交易成本**: 回测已考虑手续费，但未考虑滑点和流动性问题
4. **仅供学习**: 本工具仅用于学习和研究，不构成投资建议

## 常见问题

### Q: 如何使用真实数据？

A: 可以修改 `data_processor.py`，添加从CSV文件或API读取真实数据的功能。

### Q: 如何添加更多指标？

A: 在 `TechnicalIndicators` 类中添加新的静态方法即可。

### Q: 回测结果可以保存吗？

A: 当前版本不支持直接保存，但您可以在代码中将结果导出为CSV或Excel文件。
